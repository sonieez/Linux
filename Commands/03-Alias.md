ğŸ“Getting all the aliases (shortcut):
```bash
alias
```

ğŸ“Creating Alias:
```bash
alias name=command    #when we write this name, it will do the command's job
```
For example:
```
alias s=clear    #when we write s, it will clear
```

ğŸ“Removing Alias:
```bash
unalias name   #deletes the name from aliases (it won't work anymore)
```
